MergeCtx 类

发表于2021-12-13|更新于2022-02-28|VTM代码

简介

记录Merge候选MVP的上下文信息

成员变量

名称
类型
描述
mvFieldNeighbours MvField [MRG_MAX_NUM_CANDS << 1] 候选MVP列表
BcwIdx uint8_t [MRG_MAX_NUM_CANDS] BCW索引
interDirNeighbours unsigned char [MRG_MAX_NUM_CANDS] 候选MVP参考方向
mrgTypeNeighbours MergeType [MRG_MAX_NUM_CANDS] 候选MVP的Merge类型
numValidMergeCand int 有效MVP数量
hasMergedCandList bool 是否存在Merge候选列表
subPuMvpMiBuf MotionBuf SbTMVP子块运动信息缓冲区
subPuMvpExtMiBuf MotionBuf 未使用
mmvdBaseMv MvField [MMVD_BASE_MV_NUM][2] MMVD基础MV
mmvdUseAltHpelIf bool [MMVD_BASE_MV_NUM] MMVD基础MV是否使用半精度
useAltHpelIf bool [MRG_MAX_NUM_CANDS] Merge候选MVP是否使用半精度

成员函数

名称
描述
setMergeInfo() 将Merge模式MVP候选的运动信息复制到PU
setMmvdMergeCandiInfo() 将MMVD Merge模式初始MVP候选细化并复制运动信息到PU
