PU::getInterMergeCandidates

发表于2021-12-13|更新于2022-02-15|VTM代码

功能

建立Merge模式候选列表，其候选MVP来自于基于相邻块的空域MVP，基于同位块的时域MVP，基于历史信息的MVP(HMVP)，逐对的平均MVP和零运动矢量

输入

  • const PredictionUnit &pu：当前PU
  • MergeCtx& mrgCtx：Merge模式信息
  • int mmvdList：未使用
  • const int& mrgCandIdx：候选列表截止长度
