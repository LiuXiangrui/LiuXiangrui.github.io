PU::addMergeHMVPCand

发表于2021-12-13|更新于2022-02-15|VTM代码

功能

向Merge候选列表中添加历史MVP(HMVP)候选

输入

  • CodingStructure &cs：当前CS
  • MergeCtx& mrgCtx：Merge模式信息
  • const int& mrgCandIdx：候选列表截止长度
  • const uint32_t maxNumMergeCandMin1：Merge候选列表最大长度-1
  • int &cnt：当前Merge候选数量
  • const bool isAvailableA1：左侧相邻CU是否可用
  • const MotionInfo miLeft：左侧相邻CU的运动信息
  • const bool isAvailableB1：上方相邻CU是否可用
  • const MotionInfo miAbove：上方相邻CU的运动信息
  • const bool ibcFlag：是否启用IBC
  • const bool isGt4x4：PU尺寸是否大于4x4
