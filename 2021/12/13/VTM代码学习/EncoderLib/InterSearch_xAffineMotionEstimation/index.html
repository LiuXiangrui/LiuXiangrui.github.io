InterSearch::xAffineMotionEstimation

发表于2021-12-13|更新于2022-03-01|VTM代码

功能

以CPMVP为起点，进行运动估计，在参考帧中搜索最佳MV

输入

  • PredictionUnit& pu：当前PU
  • PelUnitBuf& origBuf：原始像素
  • RefPicList eRefPicList：参考帧列表
  • Mv acMvPred[3]：CPMVP
  • int iRefIdxPred
  • Mv acMv[3]：最佳MV
  • uint32_t& ruiBits：最佳MV对应比特率
  • Distortion& ruiCost：最佳MV对应代价
  • const AMVPInfo& amvpInfo：AMVP信息，用于整数细化
  • int& mvpIdx
  • const AffineAMVPInfo& aamvpi：仿射AMVP候选
  • bool bBi：是否为双向预测
