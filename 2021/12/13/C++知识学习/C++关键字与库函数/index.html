C++关键字与库函数

发表于2021-12-13|更新于2022-02-27|C++

Sizeof 与 Strlen 作用到字符数组到区别

  • strlen 是函数，在程序执行时计算；sizeof 是运算符，在编译时计算
  • strlen 测量字符串的实际长度，sizeof 测量字符数组的分配大小
  • 若字符数组 arr 作为函数的形参，sizeof(arr) 获得字符指针的长度，strlen(arr) 获得字符串长度
  • 前者在运行时计算字符串长度，后者在编译时计算字符数组长度。

Explicit 避免编译器进行隐式类型转换

  • explicit 关键字用来声明类构造函数是显示调用的可以阻止调用构造函数时进行隐式转换
  • explicit 关键字只可用于修饰单参构造函数

Static 的作用

static 关键字可以定义局部静态变量、外部静态变量、静态函数、静态成员变量和静态成员函数

  • static 作用于局部变量，改变了局部变量的生存周期，使得该变量存在于定义后直到程序运行结束的这段时间
  • static 作用于全局变量和函数，使得全局变量和函数只能在定义它的文件中使用
  • static 作用于成员变量和成员函数，使得成员变量或成员函数和类有关，可以通过类访问
    • 静态成员变量是在类内进行声明，在类外进行定义和初始化
    • 静态成员变量可以作为成员函数的参数，而普通成员变量不可以
    • 静态成员变量的类型可以是所属类的类型，而普通成员变量的类型只能是该类类型的指针或引用
    • 静态成员函数不能调用普通成员变量和成员函数
    • 静态成员函数没有 this 指针，不能声明为虚函数、const 函数和 volatile 函数

Const 的作用

  • const 修饰成员变量，定义为 const 常量，相较于宏常量，可进行类型检查，节省内存空间，提高了效率
    • const 成员变量只能在类内声明与定义，在构造函数初始化列表中初始化
    • const 成员变量只在某个对象的生存周期内是常量，不能在类的声明中初始化
  • const 修饰函数参数，使得传递过来的函数参数的值不能改变
  • const 修饰成员函数，定义为 const 成员函数，使得成员函数不能修改任何类型的成员变量，也不能调用非 const 成员函数

示例

  • 第一个const：函数返回值是一个指向常量对象的指针
  • 第二个const：参数p指向的对象是一个常量对象
  • 第三个const：参数p是一个常量指针
  • 第四个const：Func函数是const类型的，函数里面不能对成员变量进行修改，也不能调用非const的成员函数

Define 宏常量 与 Const 常量的区别

  • 编译阶段：宏常量是在编译预处理阶段进行替换，const 是在编译阶段确定其值
  • 安全性：宏常量只是进行简单的替换，不会进行类型安全的检查；const 常量有类型的，需要进行类型安全检查
  • 内存占用：宏常量在程序中使用多少次就会进行多少次替换，占用代码段的空间；const 常量占用静态存储区的空间，程序运行过程中只有一份
  • 调试：宏常量在预编译阶段进行替换，不能调试；const 定义的常量可以进行调试
